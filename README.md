# HR Dashboard

An interactive HR analytics dashboard created using Microsoft Excel.

## Features
- Employee statistics
- Gender distribution
- Salary analysis
- Leave tracking
- Regional employee analysis
- Employment status breakdown

## Tools Used
- Microsoft Excel
- Pivot Tables
- Charts
- Slicers
